# TYPECOUNT — Premium Character Counter

**TYPECOUNT** is a full-featured, beautifully designed character counter built for writers, editors, and students. It provides deep insights into your writing with readability analysis, word frequency, heatmaps, and more—all within a sleek, responsive interface.

## ✨ Features

- **Live Counting**: Get instant feedback on character, word, and line counts as you type.
- **Readability Analysis**: Automatic calculation of multiple readability scores:
  - Flesch Reading Ease
  - Flesch-Kincaid Grade Level
  - Gunning Fog Index
  - SMOG Index
  - Automated Readability Index (ARI)
- **Visual Analytics**:
  - Word and Letter frequency charts.
  - Interactive Heatmaps for word length and frequency.
- **Smart Tools**:
  - Sample text loader for testing.
  - Snapshot history (auto-saves every 5 seconds).
  - Export full stats reports.
  - Desktop-grade keyboard shortcuts.
- **Premium Design**: Modern aesthetic with a subtle cursor glow, glassmorphism elements, and smooth transitions.

## 🚀 Getting Started

Since TYPECOUNT is built with vanilla web technologies, there's no installation or build step required.

### Local Development
1. Clone the repository.
2. Open `index.html` in any modern web browser.

Alternatively, you can run a simple local server (recommended for the best experience):
```bash
# Using Python
python -m http.server 8080
```
Then visit `http://localhost:8080`.
